Installtion:
--------------------------------------------------------------------------------
1. Install the Google Analytics API(drupal.org/project/google_analytics_api)
   module by placing the modules files in your module directory.  Then activate
   the module at the module list page (admin/build/modules).

2. Enable the Google user account that you will be using to pull your Analytics
   data with (admin/settings/google-analytics-api).
   
3. Install the Google Analytics Importer Module by placing the module files in
   your module directory.  Then activate the module at the module list page
   (admin/build/modules).
   
4. Limit the CPU load through the settings page for the Google Analytics
   Importer(admin/settings/ga-importer).

5. Run cron.
